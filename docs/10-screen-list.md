# Screen List

List of screens for the React Native mobile application across different user roles.

## Common Screens
* **Splash Screen**
* **Login / Sign Up**
* **Role Selection** (Member, Gym Owner, Trainer)
* **Forgot Password**
* **Profile Settings**

## Member App
* **Home / Discover:** Map and list view of nearby gyms, search bar.
* **Gym Details:** Gym photos, facilities, equipment list, plans.
* **My Gym Dashboard:** Quick access to active gym, current crowd level.
* **QR Check-in:** Display QR code to scan at entry.
* **Assessment / Onboarding:** Form to enter height, weight, goals.
* **Workout Plan:** Daily exercise list, sets, reps.
* **Progress Tracking:** Charts for weight tracking, progress photos.
* **Trainer Request:** Button to call for help.

## Gym Owner App
* **Owner Dashboard:** High-level metrics (active members, revenue, daily check-ins).
* **Gym Profile Management:** Edit details, upload photos.
* **Equipment Inventory:** Add/edit/remove equipment.
* **Membership Plans:** Create and manage pricing plans.
* **Members List:** Directory of all active/inactive members.
* **Analytics & Reports:** Detailed charts (peak hours, retention alerts).

## Trainer App
* **Trainer Dashboard:** Overview of assigned members.
* **Assistance Requests:** Queue of live requests from members on the floor.
* **Member Profile:** View member's workout plan and goals (read-only or edit mode).

## Admin Panel (Web - Future/Optional)
* **Dashboard:** Platform-wide metrics.
* **Gym Approvals:** Review and approve new gym registrations.
* **User Management:** Manage platform users.
